Class: save

save(model, newValue, dispatch, typeopt)

new save(model, newValue, dispatch, typeopt)

Save an object to state, and optionally to store if persistData is true (default). Don't check if current state and new value are different so if there is an infinite loop, the developer will know right away in the console log.
